How it works

Up and running in three steps

{{ platform_name }} runs inside rootless Podman containers on any Linux, macOS, or Windows machine.

1
Install
Run a single command. The app launches in rootless Podman containers with a Caddy reverse proxy. No Docker required.
2
Set Up
A setup wizard walks you through naming your server, choosing modules, and creating your admin account.
3
Connect
Your family connects from any device on the local network. Want to share outside your home? Enable a secure tunnel — no port forwarding needed.

Technical Details

Stack
Python / FastAPI / SQLite / Jinja2 / htmx
Containers
Rootless Podman pod (Caddy + app)
Data
SQLite + flat files on your own disk
Auth
Local accounts with Argon2 hashing
Federation
ActivityPub (opt-in)
