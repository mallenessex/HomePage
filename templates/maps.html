How To Load Data

  1. Export a small OpenStreetMap area as raster tiles or an `MBTiles` file on another machine.
  2. Copy the tiles into `data/maps/tiles` or place the `MBTiles` file at `data/maps/base.mbtiles`.
  3. Refresh this page. The viewer will use local files only.

V1 Scope

This first pass is a local tile browser: pan, zoom, and inspect. Search, markers, routes, and geocoding can be added later.
